Custom Text Tutorial

Monday, October 11, 2010

 

VIDEO TUTORIAL. This is it! The text editing tutorial! It’s really a lot easier than it looks! And kind of... fun. Scary right?


The wonderful and amazing program you see above is none other than Advance-Text! It is simply the single best and greatest program (in my humble) opinion out there for beginner text editors.


So, let’s get started. Bust out your A-Text copy which I have so mindfully included in my download pack. If you receive the following notification:


It’s time to do some googling. Or look here. Download and follow the instructions on that page to install and register comdlg32.ocx.


Once all of that is out of the way, and you have A-Text up and running, simply open your ROM as you would with any of the other tools.


The ROM that you loaded may or may not activate one of the packaged INI files that tell it what the text pointers are. For this tutorial, however, we will not be using them. Fire up VBA and load up the ROM of the same name.


Take it away, me:




















  1. -While playing the game find the text that you wish to edit.

  2. -Click on Search in Advance Text

  3. -Type the text exactly as it is in the game

  4. -Check the box that says “Start from Beginning”

  5. -Click Search button

  6. -If the text on the right is what you wish to edit, hit save and give it a name and category

  7. -Edit the text accordingly with the interface
