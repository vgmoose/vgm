Included in this pack:

  1. -Attack Editor GB - Edit Attacks. Functions with all Gen I games.

  2. -Poke Edit GB - Edit the stats of Pokémon. Functions with all Gen I.

  3. -Pokemap RB - Edit the maps. Compatible fully with Red and Blue versions (USA). Limited support is available for other Gen I (see readme in folder)

  4. -Pokemon GAE4E - Gym and Elite Four editor for all Gen I. Pretty straightforward.

  5. -Pokemon Title Editor RB - Edit the Pokémon on the R/B title screen.

  6. -PokeStarter GB - Change the starter Pokémon in all Gen I games.

  7. -Poketext - Edit the text! Compatible with all Gen I and Gen II games.

  8. -RBG Title Editor - Another title screen editor. Not for Yellow version.

  9. -RBGYTMEditor - TM editor for all of Gen I.

  10. -RBTitlescreenEditor - Yet another title screen editor for R/B

  11. -RBY Mart Editor - Edit marts in all Gen I.

  12. -RBYGtrade - Edit in game trades in all Gen I.

  13. -Silent - Another mart editor. Only functions for R/B (USA).

  14. -Yellow Starter Editor - Edit all Overworld sprites in Pokémon yellow.

  15. -Visual Boy Linker - Gameboy emulator with link cable properties


A brief readme is also included containing references to anything you might need later.
